NAME

envetutorial - 首次介紹 enve

SYNOPSIS

enve *

DESCRIPTION

enve是一系列的命令工具用來協助位處於不同地點的開發者們共同維護著一致的開發環境、分享團隊中的資源與工具、管理團隊中的服務與工作

開發環境的建置風險 測試平台的建置風險 套件相依性的建置風險 部署程序的建置風險

FEATURE 特色

簡單 - 概念好理解，不使用任何複雜演算法，即使用其他語言重新實作也非常容易 模組化 - 在無任何前提下不用編譯即可相容任何平台，透過公開穩定的協定與其他軟體合作 去中央 - 不須依賴中央伺服器即可安全的完成任務

WHY ENVE 為何要用

為何要做enve 使用enve能帶來什麼好處：易學 預設值就是最直覺簡單的狀況，但不是最推薦的狀況

作為經營者，提供員工易上手 作為開發者，輕鬆分享成品 作為新語言，可採用enve作為現成的套件管理 作為服務商，免費即可enve整合入系統

使用enve能夠脫離各種平台與協定的約束 使用enve能夠擺脫中央集權的控制

INSTALL

enve不需要編譯，也不需要在系統中建立任何的永久檔案結構就能夠執行。這讓enve的安裝方式更彈性而且跨平台。

./bin/enve install enve
