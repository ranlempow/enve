NAME

enve - encourage people working together via the simplest idea

SYNOPSIS

enve command [-h|--help] [-D|-V|-Q] [--debug=debug]
     [--verbose-level=level] [--progress=progress]
     [--color=color] [--logfile=file] [--logfile-level=filelevel]
     [args]
enve -v
enve -h|--help

DESCRIPTION

enve是一系列的命令工具用來協助位處於不同地點的開發者們共同維護著一致的開發環境、分享團隊中的資源與工具、管理團隊中的服務與工作。初次使用可參見envetutorial(7)中的引導教程，或是從enveconcept(7)中開始一系列的概念介紹，也可以enve-init(1)與enve-enter(1)這幾兩個常用指令作為起點。

這裡講述的是enve的通用選項，command可以是之後小節中的命令，每個命令都可以使用這些通用選項。通用選項主要是用來調整輸出訊息，或是用來查詢命令的幫助訊息。

OPTIONS

-v
印出enve的版本訊息。
-h
印出command的簡易幫助訊息，或是enve的幫助訊息。
--help
等同於enve-help(1)，顯示較完整的幫助訊息。
-D
輸出更多的除錯資訊，可以重複使用多次。
-V
輸出更多的詳細資訊，可以重複使用多次。
-Q
輸出更少的資訊，可以重複使用多次。
--verbose-level=level
指定事紀的輸出等級，選項有error, warning, info, detail, debug, trace。
--debug=auto
預設值，產生廣泛的除錯訊息
--output-format=auto
預設值，互動時輸出整齊格式，管道時輸出標準事紀格式
--progress=auto
預設值，互動時且非必需時以一行空間重複列印文字，否則輸出完整文字
--color=auto
預設值，互動而且支援色彩時輸出有色文字，否則輸出黑白文字
--logfile=file
額外產生一份事紀到file
--logfile-level=level
指定額外事紀的輸出等級
