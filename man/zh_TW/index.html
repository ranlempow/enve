NAME

enve - encourage people working together via the simplest idea

DESCRIPTION

這裡是enve的說明文件。enve是一套用來幫助開發者們協同開發的軟體。

首次看這個文件的人可以先從envetutorial(7)來瞭解enve是怎麼回事，以及評估你該不該用enve。然而我們更推薦閱讀enveconcept(7)來了解enve的重要概念，envecodebase(7)是很好的開始。稍微了解一些詞彙之後，enveexample1(7)有一連串的小故事以個人為觀點來演示使用enve的情境。enve的每個命令都能從enve(1)找他們的詳細用法。有些重要的語法或通訊協定也收藏在enve(1)的某些命令中，我們會在enveconcept(7)的各個章節標明這種狀況。
